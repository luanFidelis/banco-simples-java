# Sistema Bancário Simples em Java

Projeto desenvolvido para demonstrar conhecimentos em Programação Orientada a Objetos utilizando Java.

## Funcionalidades
- Autenticação por senha
- Depósito de valores
- Saque com verificação de saldo
- Consulta de saldo
- Histórico de transações
- Menu interativo no console

## Conceitos Aplicados
- Encapsulamento
- Separação de responsabilidades
- Organização em pacotes (model e service)
- Uso de coleções (List)
- Estrutura modular e escalável

## Estrutura do Projeto
banco-simples/
│
├── model/
│   └── ContaBancaria.java
│
├── service/
│   └── BancoService.java
│
└── Main.java

## Como Executar

1. Compile o projeto:
   javac Main.java model/ContaBancaria.java service/BancoService.java

2. Execute:
   java Main

## Objetivo

Projeto criado com foco em prática para desenvolvimento backend Java e preparação para oportunidades de estágio.
